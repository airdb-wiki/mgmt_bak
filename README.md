# mina

> A MiniProgram project in Wechat base on [Tencent Wepy](https://github.com/Tencent/wepy)

## Wepy Install & Usage

[doc](https://tencent.github.io/wepy/document.html#/./doc.cli)

## Node Modules Install

``` bash
npm install
```

# Develop
``` bash
npm run dev
```

# build
``` bash
npm run build
```
