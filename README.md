## Wepy Install & Usage

This MiniProgram project in Wechat base on [Tencent Wepy](https://github.com/Tencent/wepy)
[wepy doc](https://tencent.github.io/wepy/document.html#/./doc.cli)

## Install & Run

``` bash
npm install
wepy build
wepy build --watch
```
